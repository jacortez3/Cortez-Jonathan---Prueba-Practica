Evaluación 1er Parcial (Parte Práctica).

Crear un repositorio en Github, vincularlo con su carpeta local, adjunte el desarrollo de cada microproyecto dentro del repositorio github, dicho repositorio compartalo mediante un documento pdf.

Ejercicio 1: Creación de Componente Simple

  1. Crea un nuevo componente web llamado saludo-component.
  2. El componente debe renderizar un saludo simple, como "¡Hola, Mundo!".
  3. Utiliza el Shadow DOM para encapsular el contenido.

Ejercicio 2: Comunicación entre Componentes

  1. Crea dos componentes web: emisor-component y receptor-component.
  2. El componente emisor-component debe tener un botón.
  3. Al hacer clic en el botón, el componente emisor-component emite un evento personalizado con un mensaje.
  4. El componente receptor-component escucha ese evento y muestra el mensaje recibido.

Ejercicio 3: Uso de Slots

  1. Desarrolla un componente llamado contenedor-component.
  2. El componente debe tener un área de encabezado y un área de contenido, utilizando slots.
  3. Permite que el contenido del encabezado y del cuerpo sea personalizado cuando se utiliza el componente.

Ejercicio 4: Integración con API Externa

  1. Crea un componente llamado user-list-component.
  2. Este componente debe obtener datos de una API externa que proporciona una lista de usuarios.
  3. Muestra la lista de usuarios en el componente.

Ejercicio 5: Gestión de Estado

  1. Desarrolla un componente contador-component
  2. El componente debe mostrar un número que puede incrementarse y disminuirse.
  3. Utiliza una variable interna para gestionar el estado del contador
  4. Proporciona botones para incrementar y disminuir el contador
